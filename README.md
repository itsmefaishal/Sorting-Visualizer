This Sorting Visualizer works on -
1. Merge Sort
2. Quick Sort
3. Bubble Sort
4. Insertion Sort
5. Heap Sort
6. Selection Sort

Color meaning -

1. Yellow - element being compared
2. Red - element identified as wrong 
3. Blue - element in its default position
4. Green - element in its correct position
